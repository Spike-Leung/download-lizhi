# Get started

需要安装 [deno](https://deno.land/)，然后执行：

```js
deno task download
```

下载的内容在 `albums/` 目录下。

下载完成后，你可以利用 [ Binaryify / NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi) 上传到网易云云盘（如果你是一个网易云用户的话 :P ）
